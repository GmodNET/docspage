Call(Int32, Int32)

Calls a function (or any other callable object) followed by its arguments from the stack. Pops the function and arguments and push function’s return values onto the stack. WARNING: this method is unsafe and should not be used in the most cases. Use MCall(Int32, Int32) and PCall(Int32, Int32, Int32) instead.

Declaration
[Obsolete("This method is unsafe. Use ILua.MCall or ILua.PCall instead.", false)]
void Call(int iArgs, int iResults)
Parameters
Type Name Description
Int32 iArgs

Number of arguments to pass to the function.
Int32 iResults

Number of the function’s return values to push back onto the stack.
Remarks

Call(Int32, Int32) is the most basic and unsafe way to call a Lua function. Unlike MCall(Int32, Int32) and PCall(Int32, Int32, Int32), Call(Int32, Int32) does not catch any errors or exceptions thrown by a callee. Thus, any thrown error can cause a complete termination of the game or server process, especially on Linux and macOS platforms.

See lua_call function in the Lua manual: https://www.lua.org/manual/5.1/manual.html

Examples

The following example function prints a message to the game console by calling Garry’s Mod native function print with Call(Int32, Int32).

static int HelloWorld(ILua lua)
{
    lua.PushSpecial(SPECIAL_TABLES.SPECIAL_GLOB);
    lua.GetField(-1, "print"); // Push Lua function to the stack
    lua.PushString("Hello World!"); // Push an argument for print function
    lua.Call(1, 0); // Call print function with one argument and zero return values
    lua.Pop(1); // ILua.Call has poped function and string from the stack, so there is only global table left

    return 0;
}

MCall(Int32, Int32)

Calls a function (or any other callable object) followed by its arguments from the stack in the protected mode (will catch any exception). All caught exceptions are rethrown as GmodLuaException. Pops the function and arguments from the stack.

Declaration
void MCall(int iArgs, int iResults)
Parameters
Type Name Description
Int32 iArgs

Number of arguments to pass to the function.
Int32 iResults

Number of the function’s return values to push back onto the stack.
Remarks

If execution is successful, pushes function’s return values back to the stack. Otherwise, nothing will be pushed, and GmodLuaException will be thrown.

This makes MCall(Int32, Int32) a better alternative to PCall(Int32, Int32, Int32), since any thrown exception then can be caught with usual C# try/catch block.

Examples

The following example shows, how to call a Lua function with MCall(Int32, Int32) and catch any possible exceptions.

public static int CallAndCatch(ILua lua)
{
    lua.PushSpecial(SPECIAL_TABLES.SPECIAL_GLOB);
    lua.GetField(-1, "print");
    lua.PushString("Lua function was called");
    try
    {
        lua.MCall(1, 0); // Calling Lua print function
    }
    catch(GmodLuaException e) // Catchs a Lua exception, if thrown
    {
        Console.WriteLine("Print function threw an exception: " + e.Message);
    }
    finally
    {
        lua.Pop(1);
    }

    return 0;
}
Exceptions
Type Condition
GmodLuaException

Lua engine exception was thrown while function execution.

Next(Int32)

Pops a key from the top of the stack, and pushes a key-value pair from the table at the given index (the "next" pair after the given key). If there are no more elements in the table, then lua_next returns 0 (and pushes nothing).

Declaration
int Next(int iStackPos)
Parameters
Type Name Description
Int32 iStackPos

The stack index of the table to get a key-value from.
Returns
Type Description
Int32

0 if there are no more key-value pairs to push, and non-zero integer otherwise.
Remarks

Allows you to iterate tables similar to Lua pairs(...). If the value on the top of the stack is nil, then the first key-value pair will be pushed onto the stack. After successful execution, the key will have stack index -2 and the value will have index -1.

See lua_next function in the Lua manual: https://www.lua.org/manual/5.1/manual.html

Examples

The following example demonstrates usage of Next(Int32) to print server players’ nicknames to the game console.

public static int TableIter(ILua lua)
{
    List<string> players = new List<string>();

    lua.PushSpecial(SPECIAL_TABLES.SPECIAL_GLOB);
    lua.GetField(-1, "player");
    lua.GetField(-1, "GetAll");
    lua.MCall(0, 1); // Will push a table of all players on server
    lua.PushNil(); // Calls ILua.Next on nil value to start iterating over table
    while(lua.Next(-2) != 0) // Iterating over players table
    {
        lua.GetField(-1, "Nick"); // Extracts a method which returns a player's nick
        lua.Push(-2); // Pushes a copy of player object to pass as a param to Nick()
        lua.MCall(1, 1);
        players.Add(lua.GetString(-1));
        lua.Pop(2); // Pops player object and nickname string from the stack, leaves corresponding key to continue iteration
    }
    lua.Pop(3);

    if(players.Count != 0)
    {
        Console.WriteLine("Players on server:");
        foreach(string n in players)
        {
            Console.WriteLine(n);
        }
    }

    return 0;
}

PCall(Int32, Int32, Int32)

Calls a function (or any other callable object) followed by its arguments from the stack in the protected mode (will catch any exception). Pops the function and arguments from the stack.

Declaration
int PCall(int IArgs, int IResults, int ErrorFunc)
Parameters
Type Name Description
Int32 IArgs

Number of arguments to pass to the function.
Int32 IResults

Number of the function’s return values to push back onto the stack.
Int32 ErrorFunc

The stack index of an error handler function.
Returns
Type Description
Int32

0 in case of success or one of the error codes (defined by Lua specification).
Remarks

If execution is successful, pushes function’s return values back to the stack. Otherwise, if exception was thrown while function call, pushes error message to the stack and returns exception’s code. If ErrorFunc is 0, then the error message returned on the stack is exactly the original error message. Otherwise, ErrorFunc is the stack index of an error handler function (must be a real index, pseudo-indexes are not supported). In case of runtime errors, this handler function will be called with the error message and its return value will be the message returned on the stack by PCall(Int32, Int32, Int32).

See lua_pcall function in the Lua manual: https://www.lua.org/manual/5.1/manual.html

SetField(Int32, in String)

Does a table key-value assignment t[k] = v, where t is the table at iStackPos, k is a string key, and v is a value on top of the stack.

Declaration
void SetField(int iStackPos, in string key)
Parameters
Type Name Description
Int32 iStackPos

Position of the table on the stack.
String key

Key in the table to set value for.
Remarks

Pops the value from the top of the stack. This method can be used to fill Lua tables with values or update them. In the example below we will add a Lua funtion SayHello to the Lua global table SPECIAL_GLOB. This function then can be called from any Lua script as SayHello().

See lua_setfield function in the Lua manual: https://www.lua.org/manual/5.1/manual.html

Examples
public int LuaFunc(ILua lua)
{
    lua.PushSpecial(SPECIAL_TABLES.SPECIAL_GLOB);
    lua.PushManagedFunction(lua =>
    {
        lua.PushSpecial(SPECIAL_TABLES.SPECIAL_GLOB);
        lua.GetField(-1, "print");
        lua.PushString("Hello!");
        lua.MCall(1, 0);
        lua.Pop(1);

        return 0;
    });
    lua.SetField(-2, "SayHello");
    lua.Pop(1); // As you can see we pop only one value from the stack (remaining global table) since the function itself was poped by SetField already

    return 0;
}

public static int SetTableExample(ILua lua)
